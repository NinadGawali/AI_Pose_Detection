## Tips for Best Results

1. **Lighting**: Ensure good lighting for better pose detection
2. **Camera**: Position yourself fully visible in the camera frame
3. **Space**: Stand at an appropriate distance from the camera
4. **Stability**: Keep the camera stable during practice
5. **Countdown**: Use the full 5-second countdown to get into position
6. **Recording**: Hold the pose steady for the full 5 seconds
7. **Review**: Check the "What Went Wrong" section after each practice

## Troubleshooting

**Camera not working?**
- Check if another application is using the camera
- Ensure camera permissions are granted
- Try restarting the application

**Excel file errors?**
- Make sure the Excel files are not open in another program
- Check file permissions in the directory

**Authentication issues?**
- Use the exact usernames and passwords (case-sensitive)
- Check that users.xlsx exists and contains the credentials

**Layout issues?**
- Try refreshing the page
- Check your browser zoom level (100% recommended)
- Clear browser cache if needed

## Customization

### Adding New Users
Edit `users.xlsx` and add rows with:
- Username
- Password
- Role (student/teacher)
- Full_Name

### Changing Recording Duration
In `student_app.py`, line ~306, modify:
```python
recording_duration = 5  # Change this value (seconds)
```

### Changing Countdown Duration
In `student_app.py`, line ~281, modify:
```python
for i in range(5, 0, -1):  # Change 5 to your desired countdown
```
